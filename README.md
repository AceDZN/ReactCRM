# ReactCRM
React/Redux/Gulp/FireBase

### Checkout

:computer: Checkout this repo, install dependencies, then start the http-server process with the following:

```
	> git clone git@github.com:AceDZN/ReactCRM.git
	> cd ReactCRM
	> npm install
	> gulp
```
